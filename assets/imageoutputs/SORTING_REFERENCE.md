## Sorting Tips

1. **Start with the easiest to identify:**
   - Very small icons (8x8)
   - Text/title images
   - Solid black screen
   - Maps and backgrounds

2. **Look for distinctive features:**
   - Submarine = horizontal shape
   - Moon = circular crescent
   - People = vertical orientation
   - Water = dark blue texture

3. **Use the unsorted folder** for images that don't match any prompt

4. **Check dimensions** if you're unsure - you can use PowerShell:
   ```powershell
   Add-Type -AssemblyName System.Drawing
   $img = [System.Drawing.Image]::FromFile("path\to\image.png")
   Write-Host "$($img.Width)x$($img.Height)"
   $img.Dispose()
   ```
