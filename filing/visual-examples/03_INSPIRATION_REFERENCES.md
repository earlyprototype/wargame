# Visual Inspiration References

## Games That Match Your Aesthetic

### 1. DEFCON (Introversion Software)
**Why it matters:** This is literally a nuclear war strategy game with the exact vibe you're building.

**Visual Style:**
- Vector graphics tactical map
- Deep blue background with bright cyan/red alerts
- Minimalist, functional UI
- Real-time threat tracking

**UI Elements:**
- Strike force indicators
- Escalation timeline
- DEFCON level display (5 → 1)
- Territory control overlays

**Color Palette:**
- Background: Deep Navy/Black
- Allies: Bright Cyan
- Enemies: Bright Red
- Neutral: Grey
- Alerts: Amber/Orange

**Reference Images:**
- Search: "DEFCON game UI screenshot"
- Key screens: Main tactical map, DEFCON level indicator, strike planning

**What to steal:**
- The colour scheme (matches your CLI)
- Minimalist data display
- Vector-based simplicity
- Threat level indicators

### 2. WarGames (1983 Film - WOPR Interface)
**Why it matters:** The archetypal Cold War computer interface.

**Visual Style:**
- Green phosphor monochrome terminal
- ASCII map of the world
- Teletype-style text output
- Blinking cursor

**UI Elements:**
- Command-line input
- Map grid coordinates
- Missile trajectory lines
- Status readouts

**Reference:**
- Search: "WarGames WOPR interface screenshot"
- Key scene: "Shall we play a game?" terminal

**What to steal:**
- Green CRT glow effect
- Typewriter text animation
- Map as ASCII/simple graphics
- Command-line interaction pattern
