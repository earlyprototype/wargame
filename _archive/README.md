# Archive

This folder contains files that are no longer part of the active project but
have been preserved for reference. Everything here was moved during a repo
cleanup on 4 March 2026.

The pre-cleanup state is captured in git commit `a6d15ba` ("Pre-cleanup
snapshot") if you need to find where anything originally lived.

## Structure

| Folder | Contents |
|--------|----------|
| `scratch/` | One-off scripts, debug tools, test outputs, logs, compiled scene JSON, batch generation artefacts, and other root-level files that accumulated during development. |
| `demos/` | Rejected CLI UI demos (`cli-rejected/`), animation library evaluation demos (`animation-lib-tests/`), and backup/snippet files from `cli/`. |
| `research/` | Everything from the former `@filing/` directory: analysis docs, collaboration specs, session chats, audio transcripts, the Google AI Studio prototype, and Perun materials. |
| `design-reference/` | Visual design references from the former `filing/` directory: inspiration boards, component mappings, shadcn research, deployment package docs. |
| `ux-planning/` | UX/UI planning documents for both CLI and WebApp: master plans, phase plans, progress reports, archived research. Includes the orphaned `04_PROGRESS_REPORTS/` that was at root. |
| `design/` | Game mechanics design documents from `Game_Mechanics/`, including superseded proposals in `_OLD_Advancing/`. |
| `unused-systems/` | Inventory of systems that were built but not yet wired into the main game loop. Useful reference for future integration work. |
| `graphics-runs/` | Timestamped output runs from the pixel-art generation pipeline. |
| `generated-images/` | Generated pixel art outputs, sorting scripts, and batch references from `assets/imageoutputs/`. |
